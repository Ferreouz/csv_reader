## How to run
You are gonna need docker to run this :)

1. Clone the repo
```bash
git clone https://github.com/Ferreouz/csv_reader
cd csv_reader
```

2. Build and start the containers
```bash
docker compose up -d
```

That's it, now the server is running. Go to http://localhost:8080/ to see
